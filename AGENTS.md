# Project Instructions

## Stack

- Laravel 12
- Vue 3 + Inertia
- Tailwind
- FPDI + TCPDF

## Purpose

Manual PDF stamping system:

- Master Copy (red)
- Controlled Copy (black master + red controlled)
- Uncontrolled Copy (black master + black controlled + red uncontrolled)

## Rules

- Do NOT change existing stamp positioning
- Do NOT rewrite working stamping logic
- Only extend functionality
- Windows-first environment

## Goal

Wrap this Laravel app into a Tauri desktop app.
